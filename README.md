# Ahmed Mostafa

### Technical Game Designer / Systems Designer & Front-End Developer

I design and build interactive systems — from game mechanics, combat, progression, economy, and UI/UX systems to modern front-end dashboards and web applications.

My work sits at the intersection of design and implementation. I focus on transforming ideas into playable, usable, and polished systems with clear structure, readable logic, and strong user experience.

## About Me

I'm a Technical Game Designer and Front-End Developer focused on building systems that feel clear, responsive, and scalable.

On the game side, I work on combat systems, progression loops, economy design, upgrade systems, UI/UX flows, player feedback, level structure, and gameplay prototyping.

On the web side, I build modern front-end interfaces, dashboards, SaaS-style flows, authentication-aware experiences, data views, and responsive portfolio-ready applications.

I care about how systems behave, how users interact with them, and how design decisions translate into implementation.

## Featured Game Projects

### LostPulse

A 2D roguelite platformer built with Unreal Engine 5 and C++.

Focused on movement, combat, enemy behavior, upgrades, relics, shop systems, and roguelite progression.

**Highlights**

- 2D side-scroller movement
- Combat system with light, heavy, charge, and air attacks
- Health and damage components
- Enemy archetypes
- Shop, relic, weapon, and upgrade systems
- Meta progression and save systems
- C++-driven UI workflow

---

### BeatTheBlock

A mobile puzzle game built in Unreal Engine 5.

Focused on board interaction, drag-and-drop mechanics, match logic, combo systems, objectives, and mobile-friendly UI/UX.

**Highlights**

- Match-3 and block puzzle modes
- Drag-and-drop interaction
- Combo and special tile logic
- Objective-based level flow
- Mobile UI and board feedback
- Android-focused development workflow

---

### Deadly Curiosity

A first-person puzzle / escape-room project built in Unreal Engine 5 with C++.

Focused on interaction, room-based progression, notebook objectives, puzzle logic, and immersive UI.

**Highlights**

- Interaction and prompt system
- Notebook / objective tracking
- Door and code puzzle systems
- Room-by-room puzzle progression
- UMG-based UI
- C++ implementation focus

---

### Roblox Zombie Tycoon

A systems-heavy Roblox tycoon prototype focused on production loops, automation, raids, rewards, and monetization structure.

**Highlights**

- Worker and manager automation systems

- Rebirth and progression structure
- Daily and playtime reward systems
