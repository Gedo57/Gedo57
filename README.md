## Game Projects

### LostPulse

A 2D roguelite platformer built around responsive movement, combat timing, relic-driven builds, active skills, enemy pressure, and meta progression.

**Systems covered:**

- 2D side-scroller movement
- Light, heavy, charge, and air attacks
- Startup / active / recovery combat timing
- Health, damage, hitbox, and hurtbox components
- Relics, weapons, shop, and upgrade systems
- Meta currency, permanent upgrades, and save flow
- C++-driven UI structure

### BeatTheBlock

A mobile puzzle project focused on board interaction, match logic, special pieces, combo resolution, objectives, and mobile UI feedback.

**Systems covered:**

- Match-3 and block puzzle modes
- Board state management
- Drag-and-drop interaction
- Rocket, bomb, and color bomb special pieces
- Combo resolution rules
- Dead-board detection and reshuffle logic
- Floating score and combo feedback

### DeadlyCuriosity

A first-person puzzle / escape-room project focused on interaction design, notebook objectives, clue discovery, room-based progression, and puzzle dependencies.

**Systems covered:**

- Interactable object system
- Prompt and inspect UI
- Notebook / objective tracking
- Code panels and locked doors
- Room-by-room puzzle progression
- C++ / UMG implementation structure

### Roblox Zombie Tycoon

A systems-heavy Roblox tycoon concept built around production loops, automation, raids, rebirths, retention systems, and social progression.

**Systems covered:**

- Multi-stage production pipeline
- Worker and manager automation
- Raid rules and protection windows
- Rebirth progression
- Daily and playtime rewards
- Leaderboards and community rewards
